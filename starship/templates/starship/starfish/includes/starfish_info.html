About the Starfish Dashboard

This dashboard uses starfish-nextflow to automate the detection and analysis of Starships in fungal genomes stored on this server.

Key Features:

  • Automatic file formatting
  • Parallel processing
  • Functional annotation
  • Element detection
  • Visualization

Required Input:

  • Genome assembly (FASTA)
  • GFF3 annotation file
  • Optional: eggNOG annotations
